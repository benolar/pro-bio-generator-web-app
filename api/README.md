# API Routes

This directory contains the backend logic and API handlers.

## Endpoints

- **generate-bio.js**: Main logic for calling Google GenAI to create bios.
- **create-checkout.js**: Integration with Flutterwave for Pro payments.
- **verify-payment.js**: Server-side verification of payment transactions.
- **webhook.js**: Webhook handler for asynchronous payment notifications.
- **admin.js**: Admin operations for user management and statistics.
